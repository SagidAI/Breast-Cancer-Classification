# 🏥 Breast Cancer Detection using Logistic Regression

This project demonstrates a binary classification approach to detect breast cancer using the UCI Breast Cancer Wisconsin (Diagnostic) dataset.

## 📊 Performance Matrix
- **Model:** Logistic Regression
- **Accuracy:** 97.37%
- **Scaling:** StandardScaler (Z-score normalization)
- **Key Outcome:** Only 1 False Negative (High Recall)

## 🛠️ Features Included
- Data Preprocessing and Scaling.
- Feature Importance Analysis.
- Visualizing decision boundaries via Scatter Plots.
- Interactive prediction function for real-world testing.

## 🚀 How to run
Simply click the **"Open in Colab"** button inside the notebook to run the code in your browser.
